## 📖 Overview

Khata Flow is a modern full-stack MERN application designed for shopkeepers and small businesses to manage customer credit (Udhaar), track transactions, monitor outstanding balances, send payment reminders, and generate professional PDF statements.

The application simplifies traditional ledger management by providing a secure and responsive digital solution accessible from any device.

---

## ✨ Key Features

### Shopkeeper Features

* Secure Authentication using JWT
* Add, Edit, and Delete Customers
* Record Credit Transactions
* Record Customer Payments
* Customer Ledger Management
* Dashboard Analytics
* WhatsApp Payment Reminders
* Email Customer Credentials
* Generate PDF Statements
* Email Statements to Customers
* Profile Management

### Customer Features

* Secure Login
* View Personal Ledger
* View Outstanding Balance
* View Transaction History
* Change Password

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/khata-flow.git
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```

### Install Backend

```bash
cd backend
npm install
npm start
```
